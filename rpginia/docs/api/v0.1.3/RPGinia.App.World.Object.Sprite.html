    RPGinia.App.World.Object. Sprite

    Game object type for creating sprites.

    Constructor

    new Sprite(settings)

    Source:
    • ObjectTypes/Sprite.js, line 44
    Parameters:
    Name Type Description
    settings Object object settings.
    Name Type Attributes Default Description
    name String object name.
    settings Object object settings for only this type.
    Name Type Attributes Default Description
    spriteSheetIndex Number sprite sheet number.
    spriteIndex Number sprite number.
    frameIndex Number <optional>
    		 settings.frameFrom frame number. Works only if current sprite have the frame list. If frameFrom is not defined - setting up value to 0.
    isRepeating Boolean <optional>
    		 false should animation repeating after it's ending (Works if current sprite have frame list).
    isPlaying Boolean <optional>
    		 false should animation playing (Works only if current sprite have frame list).
    interval Number <optional>
    		 60 interval of changing animation frames.
    frameFrom Number <optional>
    		 0 frame from which the animation should be played.
    frameTo Number <optional>
    		 frame list length frame to which the animation should be played.
    coords Array.<Number> object coordinations. First value - x coord, second value - y coord, third value - width, fourth value - height. If width and/or height are not defined - they will be taken from the sprite sheet.
    type String object type. For this object type it is "sprite".
    layer Number <optional>
    		 1 object layer.
    isVisible Boolean <optional>
    		 true show object in the playground.
    Example
    // spriteSheets.json
{
 "file": "path/to/your/sprite/sheet/image.png",
 "sprites" {
  {
    "name": "test_sprite_0",
    "rect": [0, 0, 506, 287] 
  },
  {
    "name": "test_sprite_1",
    "rect": [506, 0, 504, 287] 
  }
 }
}

// game.js
const engine = new RPGinia();
const app = new engine.App();
const world = new app.World();

world.initialize({
 // Your options here...
});

// createElement method from World class can create an object with indicated object type.
world.createElement({
 name: 'Test sprite',
 type: 'sprite',
 settings: {
   spriteSheetIndex: 0,
   spriteIndex: 1
 },
 coords: [10, 34, 56, 21]
});
