    RPGinia.App.World.Object. Rectangle

    Game object type for creating rectangles.

    Constructor

    new Rectangle(settings)

    Source:
    • ObjectTypes/Rectangle.js, line 28
    Parameters:
    Name Type Description
    settings Object object settings.
    Name Type Attributes Default Description
    name String object name.
    settings Object object settings for only this type.
    Name Type Attributes Default Description
    fill String <optional>
    		 "#000000" object filling.
    coords Array.<Number> object coordinations. First value - x coord, second value - y coord, third value - width, fourth value - height.
    type String object type. For this object type it is "rectangle".
    layer Number <optional>
    		 1 object layer.
    isVisible Boolean <optional>
    		 true show object in the playground.
    Example
    // Creating a simple rectangle.
const engine = new RPGinia();
const app = new engine.App();
const world = new app.World();

world.initialize({
 // Your options here...
});

// createElement method from World class can create an object with indicated object type.
world.createElement({
 name: 'Test rectangle',
 type: 'rectangle',
 settings: {
   fill: 'red'
 },
 coords: [10, 34, 56, 21]
});
